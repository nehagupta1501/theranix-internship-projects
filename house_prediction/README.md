# House Price Prediction Using Machine Learning

## Project Description

This project predicts house prices based on the area of the house using the Linear Regression algorithm.

## Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-Learn

## Machine Learning Algorithm

- Linear Regression

## Features
 
- Load dataset
- Train machine learning model
- Test the model
- Predict house prices
- Display accuracy
- Visualize data with graph

## Dataset

The dataset contains:

- Area (Square Feet)
- House Price (Indian Rupees)

## How to Run

Install libraries:

pip install -r requirements.txt

Run:

python house_price_prediction.py
