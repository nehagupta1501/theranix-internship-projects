# Exploratory Data Analysis Project

## Objective

Perform Exploratory Data Analysis (EDA) on the Titanic Dataset.

## Libraries Used

- Pandas
- Matplotlib
- Seaborn

## Features

- Data Loading
- Dataset Information
- Missing Value Analysis
- Statistical Summary
- Correlation Heatmap
- Age Distribution
- Survival Count
- Passenger Class Distribution
- Fare Distribution

## Conclusion

EDA helps identify patterns, relationships, and important insights before building Machine Learning models.
